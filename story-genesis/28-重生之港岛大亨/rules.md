# Rules: 重生之港岛大亨

## Setting
1997-1998年的香港，回归前后。核心地点：北角唐楼劏房（陈远鸣起点）、春秧街"永信证券"（梁伯韬的老铺）、铜锣湾希慎道写字楼（远鸣资本总部）、上环德辅道中168号商厦（第一个收购物业）、深水埗制衣厂（母亲工作地）、莲香楼（茶楼谈判）、天星码头、维多利亚港。时代氛围：回归喜庆与金融风暴前夕的微妙不安交织。街景充满港式细节——叮叮车、红白蓝胶袋、凉茶铺、南北行海味铺、霓虹招牌。

## Characters
- **陈远鸣** -- 主角。肉体21岁（港大经济系三年级），灵魂67岁（2024年跨年夜心脏骤停死亡后重生回1997年）。上一世身家340亿港币，福布斯香港富豪榜第十九位，但三段失败婚姻、与子女疏离。这一世目标：用前世四十六年的记忆在三年内建立商业帝国，同时珍惜家人。性格：表面平静如水，内心笃定到近乎冷酷。说话不卑不亢，气场远超年龄。弱点：对母亲和妹妹的感情是唯一能动摇他的东西。重生的恐惧：不确定这颗年轻的心脏是否会在某个节点"记起"它本该停止跳动。
- **陈月华** -- 陈远鸣的母亲。深水埗制衣厂车衣女工，月薪六千多。丈夫1994年肝癌去世（生前炒地皮欠高利贷）。勤劳、唠叨、节俭。上一世2003年SARS期间感染去世——这是陈远鸣最大的遗憾。这一世他给了她两百万让她辞工，她坚持住七百呎的房子，不肯搬半山。
- **陈远芳** -- 陈远鸣的妹妹，17岁，圣保禄中学中五学生。成绩好，读《傲慢与偏见》。上一世嫁去加拿大，兄妹疏远。这一世陈远鸣决心供她读大学。
- **梁伯韬** -- 永信证券老板。50多岁，北角土生土长的潮州人。秃顶、啤酒肚、白衬衫口袋别两支笔（红色圈马经、蓝色写单据）。做了三十年股票经纪，经历过73年和87年股灾。一批老客户是巴士司机、的士佬、茶餐厅老板。对陈远鸣从"看不起穷学生"到"震惊"到"敬畏"的态度转变是重要的情感线。
- **杜志恒** -- 远鸣资本COO。32岁，港大法律系毕业+英国MBA，前汇丰银行企业融资部高级经理。高大、方脸、银框眼镜、Hugo Boss西装。务实、能干，精通金融结构和银行关系。是陈远鸣的"杠杆"——负责把钱变成架构、把架构变成帝国。
- **林逸晨** -- 远鸣资本CTO（即将到任）。1975年生，中大计算机系毕业。在硅谷Inktomi做分布式计算工程师。上一世2003年回港创业做电子支付，技术领先但时代太早，2006年失败。陈远鸣要在正确时机把他找回来做金融科技。技术宅，守时，被"做别人做不到的事"打动。
- **肥佬黎（黎国辉）** -- 北角房东，四十几岁，矮胖，金链金牙墨镜佛珠。拥有七栋唐楼。上一世2003年SARS后破产，搬去天水围洗碗。
- **周锦年** -- 港大经济系学长，25-26岁。在百富勤做分析师。金丝眼镜、Ralph Lauren、啫喱水、欧米茄手表。自大、势利。百富勤1998年1月破产清盘时将失业。
- **何兆丰** -- 丰隆地产老板，60出头。上环地头蛇，翡翠戒指、卡地亚袖扣。试图截胡168号物业但被陈远鸣说退。高杠杆经营，未来几年将面临困境。
- **阿珍（林宝珍）** -- 远鸣资本第一个员工，46岁，前英资证券行行政主管，金融风暴后被裁。月薪两万。
- **天星码头的风衣老人** -- 神秘人物。在股灾之夜的天星码头以异常的平静看报纸上的恒指暴跌新闻。步伐笃定，"像一个知道终点在哪里的人"。陈远鸣在上一世从未见过此人。暗示可能还有其他重生者或时间旅行者。

## Story Rules
- **类型**：重生爽文/金融商战。节奏明快，爽点密集（打脸、降维打击、先知般的预判），但不失人物深度和时代质感。
- **叙事视角**：第三人称，紧贴陈远鸣视角。粤语对白保留原汁原味（繁体/口语化），叙述部分用标准中文。
- **核心机制**：陈远鸣拥有2024年以前的完整历史记忆，包括金融事件、政治变局、科技趋势。但记忆不是硬盘——存在模糊和误差的可能性（六合彩号码的记忆焦虑是一个例子）。他的操作必须遵循1997年的客观条件（通讯技术、金融工具、法律框架）。
- **打脸公式**：每一次"降维打击"必须有铺垫和后果。对手要有合理的自信（肥佬黎的七栋楼、周锦年的百富勤背景、何兆丰的三十年经验），陈远鸣的反击要精准到位但不夸张。
- **情感锚点**：母亲和妹妹是陈远鸣的软肋和动力。每一章至少要有一个家庭场景来平衡商战的冷硬。"有些东西比金钱更重要"是贯穿始终的主题。
- **避免**：不要让陈远鸣变成无所不能的神——他的心脏隐患（重生是否有时限？）和记忆的不确定性是必须保留的悬念。不要忽略时代细节——传呼机、公共电话亭、56K拨号上网、VCD、叮叮车、红双喜香烟，这些都是1997年香港的质感。

## Plot Direction
- **近期时间线**（已完成）：六合彩中奖（6300万）→ 恒指认沽权证建仓 → 10月股灾平仓（净利润3.75亿）→ 远鸣资本成立 → 收购168号商厦 → 杜志恒入伙 → 联系林逸晨（即将回港）。
- **下一阶段**（1998-2000年）：林逸晨回港建立金融科技团队；科网泡沫期间投资硅谷公司（Ariba、Check Point）并在2000年初套现；继续低价收购商业地产和物流仓储。
- **中期目标**（2003年）：SARS期间楼市触底全面扫货；远鸣置业上市准备；保护母亲不重蹈上一世SARS感染的命运。
- **悬念线**：天星码头的风衣老人——他是谁？是否也是重生者？他是盟友还是对手？陈远鸣的心脏——重生是否有时限或代价？记忆的准确性——随着他改变历史，未来的走向是否会偏离他的记忆？
- **人物发展**：梁伯韬从小经纪到陈远鸣的忠实伙伴；杜志恒从怀疑到信任到成为帝国核心；周锦年百富勤破产后的命运（可能成为盟友或对手）。
