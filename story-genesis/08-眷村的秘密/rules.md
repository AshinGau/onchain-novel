# Rules: 眷村的秘密

## 设定
- **时间跨度**：1949年（青岛码头撤退）至2019年冬天（台北），横跨七十年。主线叙事在2019-2020年。
- **地点**：台北（三军总医院、大安区、眷村旧址）、台中（民生路147巷）、高雄（左营军港、恒义集团）。
- **社会背景**：台湾眷村文化——竹篱笆平房、各省口音混杂、老兵的思乡与沉默、"明年就反攻大陆"的集体幻觉逐渐消散。眷村改建后的物理消失与记忆留存。军系企业与政商关系。
- **氛围**：跨代沉默压抑的重量感。每一代人都背负着上一代未说出口的秘密。冬雨、医院消毒水味、铁盒里泛黄的纸张、金门高粱的辛辣。

## 角色
- **赵正阳**（28岁）：主角。赵德昌之孙、赵建国之子。科技公司后端工程师，思维方式如同debug——拆解问题、追踪root cause、建立数据模型。安静但倔强，温和但不软弱。父亲在他八个月大时"溺水"身亡，由爷爷奶奶带大。手指焦虑时在膝盖上敲击如打键盘。用加密磁碟映像保存调查资料，建TODO.md列待办清单。
- **赵德昌**（89岁，已故）：赵正阳的爷爷。山东临沂人，1949年十九岁时被拉壮丁撤退到台湾。眷村里最安静的人——帮邻居修篱笆、送红烧肉、从不解释自己沉默的原因。一辈子活在对孙守义背叛的困惑与对儿子之死的愧疚中。临终前将七十年的秘密全部告诉了孙子。1973年曾独自去高雄看过孙志恒，看到他抱孙子叫"阿公"后转身离开——这件事他对任何人都没说过。
- **赵建国**（1967-1991，已故）：赵正阳的父亲。海军少校。活泼、胆大，与父亲性格截然相反。在左营基地服役时发现港口走私行为，追查到恒义集团和孙志恒。联系了陈水德查证。在笔记中记录了详细的走私证据（日期、时间、船名、车牌、港区地图）。1991年9月22-23日在高雄港第三号泊位"意外溺水"身亡。验尸报告显示血液酒精浓度0.08%——但他几乎不喝酒。
- **孙守义/孙志恒**（92岁）：赵德昌在74师的班长。山东济南人。1949年撤退时带着十六根金条消失（陈水德说他"杀了人"），换名到高雄，用金条起家创立恒义集团。从五金行做到百亿营建帝国。2015年退休。仍住在高雄左营区独栋别墅。给赵德昌的告别式送了花圈——说明他一直知道赵德昌的存在。1949年分别前把最后半块馒头分给赵德昌——这个细节使他的形象更加复杂。
- **孙伟哲**（51岁）：孙志恒之子，恒义集团现任董事长。台大法律系、哈佛MBA。娶高雄政治世家之女。将恒义从传统营建扩展到土地开发、都更和绿能。刚宣布180亿高雄港周边都更案。是否知道父亲的过去？是共犯还是不知情者？
- **陈水德/陈胖子**（中风卧床）：74师同连弟兄，河北人。退伍后在台中开面馆。赵建国曾拜访他查证孙守义的事。之后恒义集团派黑衣人"问候"，他三天内转让面馆、搬家、切断所有联系。女儿陈秀兰透露：陈水德曾说"守义那时候杀了人，就为了那批东西"以及"那件事不该碰。谁碰谁死。"
- **陈秀兰**（中年）：陈水德之女。独自照顾中风父亲。母亲早逝。知道一些关于孙守义和赵建国来访的片段信息。态度中有警惕、犹豫和积压已久的情绪。
- **王淑芬**（已故，2016年）：赵德昌之妻，安徽人。赵正阳的奶奶。温和善良，赵正阳由她带大。
- **林美琴**：赵建国之妻，赵正阳的母亲。高雄本地人。赵建国死后独自带孩子两年，因经济压力将赵正阳交给公婆。后改嫁台南。透露：赵建国出事当晚十点接了一通电话后出门再未回来；葬礼上一个穿着体面的陌生人给了三十万"慰问金"并说"好人不该问太多问题"。

## 故事规则
- **类型**：跨代悬疑/历史复仇。台湾眷村背景的家族秘密。
- **叙事视角**：双线交织——现代线以赵正阳为视角中心（第三人称有限），历史线以赵德昌的回忆和笔记为载体。历史线可以展开赵德昌不在场的事件（如青岛码头的全景），但现代线必须严格限定在赵正阳能获得的信息范围内。
- **语言风格**：写实、克制、情感内敛。眷村段落充满生活细节（各省口音的争吵、酸辣粉、金门高粱、老李头的河南梆子）。现代段落则融入程序员的思维隐喻（debug、root cause、堆栈追踪、编译器警告）。对白自然，混杂国语、台语、方言。
- **节奏**：历史线缓慢、沉重、充满沉默的重量；现代线较为紧凑，赵正阳的工程师思维让调查更高效但也更危险——他动得比爷爷快，但对手也在看着。
- **避免**：不要简化孙守义为纯粹的恶人（半块馒头的细节、抱孙子的温柔）。不要让赵正阳变成动作英雄——他是程序员，不是侦探。不要忽略监视的威胁感（医院的黑风衣、殡仪馆的黑轿车、凌晨的短讯）。不要丢失眷村文化的温度和质感。
- **核心主题**："沉默的人碎起来是没有声音的。"三代人的故事是关于沉默的代价——爷爷沉默了一辈子，父亲打破沉默就死了，孙子必须决定是否继续。

## 情节走向
- **赵正阳已决定前往高雄**。高铁台北到左营一小时三十四分钟。下周五请年假。
- **核心悬念**：
  1. 1949年青岛码头，孙守义杀了谁？为什么？（零线索）
  2. 赵建国溺水前五天（9月18-22日）发生了什么？那通导致他深夜出门的电话是谁打的？
  3. 陈水德还知道多少？他的"不干净"——他在1949年也分到了东西——具体指什么？
  4. 孙伟哲是否知道父亲的过去？他是参与者还是被蒙在鼓里？
  5. 赵正阳已被监视——医院黑风衣、殡仪馆黑轿车、凌晨的匿名短讯"希望您也是明白人"。他的安全已经受到威胁。
- **结构张力**：恒义集团180亿高雄港都更案——又是高雄港，那个吞噬了赵建国的地方。七十年后同一个家族仍在掌控同一个地方。赵正阳要揭开的不只是一桩旧案，而是一个从1949年延续至今的权力结构。
