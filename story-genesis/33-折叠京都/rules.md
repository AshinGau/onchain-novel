# Rules: 折叠京都

## Setting

### 时间
2038年，空间折叠技术民用化后的第七年。

### 地点
京都——世界上第一座完成民用空间折叠的城市。城市被折叠为七个维度层：

| 层级 | 名称 | 特征 | 人口 |
|------|------|------|------|
| A层 | 表层京都 | 传统古都，旅游观光，文化保护区，控制组 | ~150万 |
| B层 | 折叠京都 | 高密度移民社会，工业化，多语种混居，有限资源实验 | ~47万 |
| Ω层(第三层) | 里京都 | 自发形成，永恒黄昏，自治聚落，无政府状态观察 | ~1,247人 |
| 第四层 | 理想城市 | 高度规划，极简美学，完全监控，理想社会模型 | 未知 |
| 第五层 | 匮乏之城 | 极端资源匮乏，生存压力测试 | 未知 |
| 第六层 | 静默之城 | 信息隔离，文化同质化实验 | 未知 |
| 第七层 | 未知层 | 数据缺失，最深层，尚未探索 | 未知 |

### 核心技术概念
- **空间折叠**：通过拓扑压力将三维空间展开为多个并存的维度层，每层占据同一宏观坐标但维度上分离。
- **维度间距**：衡量两个维度层之间"厚度"的物理量，单位为普朗克折叠单位(PFU)。安全阈值为0.70 PFU。
- **维度结节**：当间距低于0.50时，两个层的空间拓扑在局部纠缠，形成可见的光团。
- **裂缝**：维度层之间自发形成的非授权通道，位置不稳定，每2-3个月迁移。
- **拓扑锚点**：维持折叠结构的支撑点。分人工(维度稳定器阵列)和天然(空间自愈形成)两种。
- **折叠不可逆性**：已折叠的空间不能展开。若失去拓扑压力，所有层将坍缩至同一平面，层内物质重叠——即灾难性毁灭。
- **维度特征值**：每个人在折叠空间中的唯一标识。同一人在不同层的版本共享相同的维度特征值。

### 折纸公司 (Origami Corporation)
- 2031年完成京都首次城市级折叠
- 与京都市政府签有五十年独占管理协议
- 控制全部折叠硬件(维度稳定器、折叠闸门、监控系统)
- 2033年起秘密将各折叠层用作社会实验场
- 利用折叠不可逆性作为护身符：任何试图关闭系统的行为都将导致全层坍缩
- 对外维持"两层折叠"的公开叙事，隐瞒第三层以下的存在

## Characters

### 林空良 (Lin Sora) —— 主角
- 29岁，女，中日混血(母亲中国人，父亲日本人)
- A层居民，折纸公司三级折叠维护工程师
- 性格：细致观察、冷静分析、内心柔软但行动果断
- 核心创伤：母亲"死亡"(实为坠入Ω层)
- 弧线：从被动的体制执行者 → 发现真相 → 主动反抗者
- 七个维度版本的"原始版"，需要找到并联合其他六个自己

### 林美华 —— 空良的母亲
- 52岁，中国籍
- 原为B层中文书店店主
- 2035年因折叠闸门事故坠入Ω层，被官方宣布死亡
- 在里京都重新建立书店，成为社区的文化核心
- 性格：温暖但现实，有政治直觉，理解体制的力量

### 川端彻 (Kawabata Toru / K.T.) —— 折师
- 约70岁，日本人
- 折纸公司创始团队核心成员，折叠技术的真正理论奠基者
- 2033年因反对社会实验被放逐至Ω层
- 五年来独自研究多层折叠原理，发现了天然拓扑锚点
- 掌握解除折纸公司控制权的理论方案，但需要七个同源维度版本同时操作
- 性格：学者型，内敛，用折纸比喻一切

### 另一个空良 (第四层版本)
- 29岁，与主角同源但人生路径不同
- 折纸公司七级高级研究员(左眼下有疤痕——19岁实验室事故)
- 父亲因车祸去世(与主角版本的母亲失踪对称)
- 从公司内部发现社会实验真相，选择叛离
- 性格：比主角更冷硬、更决断、更像领导者
- 已找到四个版本的空良(含自己和主角)，还差三个

### 田中启介 —— 空良同事
- A层折叠维护工程师，空良的日常搭档
- 性格随和、不敏感，代表"不质疑体制"的普通人视角
- 可能在后续成为盟友或被折纸公司利用

## Story Rules

### 叙事规则
1. **始终保持空间折叠的"触感"描写**——折叠不是抽象概念，而是有温度、有声音、有质感的物理体验。每次维度过渡都要有独特的感官描写。
2. **每一章必须以悬念结尾**——读者在章末必须产生"接下来会发生什么"的强烈冲动。
3. **京都的传统美学与高科技基础设施形成持续对比**——寺庙与维度稳定器、竹林与折叠闸门、茶道与维度监测。
4. **情感线索优先于科技设定**——折叠技术为故事服务，而非反过来。核心情感：失去与重逢、身份认同、自由意志。
5. **多语言环境**——B层和更深层的对话中应出现中文、日语、越南语等多语种元素，体现折叠城市的多元性。
6. **不同层的物理特征必须一致**——A层(正常)、B层(灰霾、工业味)、Ω层(永恒黄昏、琥珀色地面)、第四层(蓝色光、乳白地面、风铃声)等，每层有固定的感官标记。

### 世界观约束
1. 折叠一旦完成不可逆，这是核心约束，不可打破。
2. 折纸公司控制所有人工拓扑锚点，这是其权力的物理基础。
3. 空间可以自发生成天然拓扑锚点，这是川端方案的科学基础。
4. 同一人在不同层的版本共享维度特征值，这是"七个空良"计划的物理前提。
5. 裂缝是自然现象，折纸公司无法完全控制，这是层间联络的可能性基础。
6. 折叠闸门"死亡"事件实为坠入深层——已知47例，实际可能更多。

### 禁止事项
- 不得引入时间旅行（折叠是空间性的，不是时间性的——川端所说的"时间性"是指不同层的社会时间流速不同，但物理时间是同步的）
- 不得让折叠变为可逆——这会破坏核心张力
- 不得让任何角色拥有"超能力"——一切行为必须在折叠物理学的框架内
- 不得简化折纸公司为纯粹邪恶——他们的动机(寻找最优城市形态)有其合理性，只是手段不道德
- 不得忽视普通居民视角——B层47万人、里京都1247人的生活细节是世界观可信度的基础

## Plot Direction

### 已完成的情节线 (第1-3章)
- [x] 空良发现维度异常读数的螺旋模式
- [x] 空良进入B层，发现B层整体间距持续偏低
- [x] 空良在中继枢纽发现Ω层的存在及折纸公司的隐瞒
- [x] 空良在里京都与"已故"母亲重逢
- [x] 空良了解47起"折叠闸门死亡"的真相
- [x] 空良在第四层遇见另一个版本的自己
- [x] 核心计划揭露：需要七个空良同时操作天然拓扑锚点
- [x] 折纸公司搜索系统开始追踪

### 未来情节方向 (第4章起)
- [ ] 寻找第五、六、七层的三个空良版本
  - 第五层(匮乏之城)的空良可能不愿合作——她的世界让她变得极端务实或绝望
  - 第六层(静默之城)的空良可能已被完全同化，失去了个人意志
  - 第七层的空良身份成谜——可能是计划的关键，也可能是最大的障碍
- [ ] 折纸公司的反应升级：从被动搜索到主动出击
  - 可能封锁层间裂缝
  - 可能对已知的空良版本实施隔离
  - 可能揭示公司内部也有分歧——不是所有高层都支持社会实验
- [ ] 川端彻的完整方案及其风险
  - 天然拓扑锚点转移的具体操作方式
  - 可能的副作用：层间边界重构可能改变各层的物理特征
  - 对里京都等已适应现有环境的社区的影响
- [ ] 母亲的选择——回到A层还是留在里京都
- [ ] 七个空良的"同步操作"场景——小说的高潮
  - 每个空良需要在自己的层做出个人牺牲
  - 操作成功后各层的关系如何重塑
- [ ] 最终主题：你无法选择自己生活在哪个"层"，但你可以选择如何对待同一座城市里的其他层

### 核心悬念
1. 第七层到底是什么？为什么数据缺失？
2. 折纸公司是否知道川端的计划？
3. "七个空良"是否都愿意合作？如果有一个拒绝呢？
4. 折叠系统控制权转移后，京都会变成什么样？
5. 空良最终会回到A层的日常生活，还是选择另一种存在方式？
